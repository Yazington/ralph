# Additional Tools and Dependencies

## Validation
- Zod: schema validation for TypeScript, ensures data integrity.

## Date Manipulation
- date-fns: lightweight date utility library for parsing, formatting, and manipulating dates.

## Recurrence
- rrule.js: library for working with recurrence rules for calendar dates based on iCalendar RFC.

## State Management
- Zustand: simple state management for React (optional, could use React Context).

## UI Components
- shadcn/ui: already included in tech stack, need to set up with Tailwind.

## Testing
- Vitest: already included.
- Playwright: for end-to-end testing.
- @testing-library/react: for component testing.
- @testing-library/user-event: for user interaction simulation.

## Data Persistence
- idb-keyval or localForage: simplified IndexedDB wrapper.
- PouchDB (optional): for sync with remote CouchDB.

## Form Handling
- React Hook Form: performant forms with validation.

## Icons
- lucide-react: icon library.

## Code Quality
- ESLint with TypeScript plugin.
- Prettier for code formatting.

## Build Tools
- Vite: already included.
- TypeScript: already included.
- Tailwind CSS: already included.

## Deployment
- Vercel or Netlify for hosting.
